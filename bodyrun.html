BODYRUN

Aplicación web para el bienestar alimenticio y rutinas físicas

Servicios Desarrollo Frontend
Tecnologías HTML, CSS, JavaScript
Rol Desarrollador Principal
Año 2023 - 2024
INFO

BodyRun es un sitio web diseñado para promover un estilo de vida saludable. Permite a los usuarios calcular su Índice de Masa Corporal (IMC), establecer metas de déficit o superávit calórico, generador de dietas, y acceder a rutinas de ejercicio personalizadas según su nivel de actividad física. El objetivo principal fue crear una interfaz intuitiva y fácil de usar que motivara a los usuarios a alcanzar sus metas de bienestar.
